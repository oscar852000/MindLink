# 验证清单

**文档维护声明**：本文档可以修改。如果发现内容与实际不符、过时、或有错误，必须修改。但保持精简，避免臃肿。

---

## 后端改动

改完后端代码后，必须确认：

- [ ] 服务能启动：`systemctl status mindlink`
- [ ] API 能调通
- [ ] 日志没有报错：`journalctl -u mindlink -f`

---

## 前端改动

改完前端代码后，必须确认：

- [ ] 桌面端页面能打开
- [ ] 移动端页面能打开
- [ ] 核心功能可用
- [ ] 控制台没有报错

---

## 数据库改动

如果涉及数据库变更：

- [ ] 迁移脚本能执行
- [ ] 旧数据不丢失
- [ ] 查询性能无明显下降

---

## 服务管理命令

```bash
systemctl status mindlink    # 查看状态
systemctl restart mindlink   # 重启服务
journalctl -u mindlink -f    # 查看实时日志
```
